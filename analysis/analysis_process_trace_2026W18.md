## 5. 分析の限界・注意事項

### 5-1. データの制約

```
[制約1] 決算データ
  - 味の素: Q3累計（2026/2/5発表）が最新。通期は予想値。
  - MJC: 推定値ベース。直近の決算短信未確認。
  → 解決策: 決算発表後の再評価が必要

[制約2] テクニカル指標
  - 移動平均線の正確な計算値は取得困難（概算値）
  - RSI, MACDの正確な数値はリアルタイムAPIが必要
  → 解決策: リアルタイムデータソースでの検証推奨

[制約3] アナリストレーティング
  - ログイン必須の情報源があり、完全な取得は不可
  → 解決策: 証券レポートの直接参照推奨

[制約4] マクロ指標の最新性
  - 2026/4/30時点のスナップショット
  - 5/1以降の指標発表で前提が変わる可能性
  → 解決策: イベント駆動型の更新プロセスが必要
```

### 5-2. モデルの不確実性

```
[不確実性要因]
1. シナリオ確率は主観的判断を含む
   → 複数アナリストの意見を加重平均すべき
   
2. スコアリングの重み付け（25:20:20:20:15）は経験則
   → バックテストによる最適化が望ましい
   
3. マクロ要因の相互作用が非線形
   → 単純な加重和では捕捉できない複雑な効果

4. ブラックスワン事象（中東急変等）は確率モデル外
   → ストレステストの実施を推奨
```
